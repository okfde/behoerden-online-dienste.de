Datenquellen

Bundesamt für Statistik

Das Bundesamt für Statistik veröffentlicht mit der GV100 alle drei Monate eine Übersicht über alle politischen Ebenen in Deutschland inklusive ihrer Regionalschlüssel, welcher sich hervorragend als ID eignet, um Daten aus anderen Quellen zuzuordnen. Mit enthalten sind außerdem einige Metadaten wie z.B. Daten zur Bevökerung.

Sämtliche Namen und die Zuordnung zu den Regionalschlüsseln auf dieser Seite stammen von der GV100. Diese kann man sich auf den Seiten des Bundesamtes für Statistik downloaden.

WikiData

WikiData ist eine Initiative innerhalb der Wikipedia, welche aus den einzelnen Feldern der Wikipedia-Artikel strukturierte Daten macht. Diese können dann über den WikiData Query Service mit SPARQL Queries abgerufen werden.

Sämtliche URLs zu den Websites stammen von WikiData und unterliegen so der Creative Commons Attribution-ShareAlike 3.0. Die Daten können mit dem Query für RGS sowie dem Query für die AGS abgerufen werden.

Online-Check der Grünen NRW

Die Landtagsfraktion von Bündnis 90 / Die Grünen haben im Jahr 2014 und im Jahr 2016 einen Online-Check durchgeführt, bei dem eGovernment-, Daten- und andere Angebote von Kommunen bewertet wurden. Ein Teil der Kommunen wurde von der Open Knowledge Foundation professionell erstellt.

Sämtliche URLs zu weiteren Diensten abseits der Website stammen aus dieser Quelle.

OpenSource: Mitmachen!

Die Seite basiert auf dem Python-Microframework Flask, MySQL und ElasticSearch. Der Code steht unter der BSD-3-Clause Lizenz.

Auf Github kann der Code eingesehen werden. Verbesserungsvorschläge und Patches sind herzlich willkommen!
